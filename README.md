# Sales Analytics Dashboard — Jan 2024 to Dec 2024

## Project Overview
A sales analytics project analysing 500+ orders across five regions, 
six product categories, and twelve months. The project covers data 
cleaning, pivot table analysis, and a KPI summary dashboard — 
built entirely in Microsoft Excel.

## Dashboard Highlights
- KPI cards showing Total Revenue (₹8,70,762), Total Units Sold 
  (5,364), Average Order Value (₹1,755), and Top Region (West)
- Revenue breakdown by Month, Quarter, Region, and Category
- Built entirely in Microsoft Excel using Pivot Tables and 
  linked KPI cards

## Key Business Questions Answered
1. Which month and quarter generated the highest revenue?
2. Which product category contributes most to total revenue?
3. Which region is the top performer by revenue?
4. Which individual products drive the highest sales value?

## Key Insights
- Q4 (Oct–Dec) generated the highest revenue at ₹3,12,472 — 
  nearly double Q3
- Electronics is the top category at ₹2,86,551 in revenue
- West region leads all regions with ₹1,98,436 in total revenue
- Laptop Pro is the single highest-revenue product

## Tools Used
